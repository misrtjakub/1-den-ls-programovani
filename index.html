Co si pod slovem programování představíte vy?

CO JE PROGRAMOVÁNÍ?

  • Proces tvorby instrukcí pro počítač

CO JE C#?

  • Víceúčelový, kompilovaný, objektově orientovaný jazyk
  • Vyvinutý v Microsoftu (2000)
  • Součást platformy .NET

PROČ C#?

  • Snadno čitelná a konzistentní syntaxe
  • Široké využití - desktop, web, hry (Unity)
  • Silná komunita a podpora

HISTORIE C# - PŘEHLED

Od první verze v roce 2000 až po .NET 8 (2024)

2000-2005

  • 2000: Oznámení C# a .NET Framework 1.0
  • 2002: C# 1.0 - první verze
  • 2005: C# 2.0 - šablony (<>), iterátory (yield)

2006-2012

  • 2007: C# 3.0 - LINQ, zkrácený zápis (=>, var)
  • 2010: C# 4.0 - dynamické typy (dynamic)
  • 2012: C# 5.0 - asynchronní kód (async/await)

2015-2024

  • 2015: C# 6.0 - interpolace řetězců ($"")
  • 2017: C# 7.x - podmínky v patternu (is, switch)
  • 2020: C# 9.0 - záznamy (record)
  • 2021-2024: C# 10-13 - .NET 6-8 (global using, list patterns)

Struktura programu

1. Import knihoven

using System;
  • Importuje knihovnu System
  • Zahrnuje základní funkce jako Console

2. Třída programu

class Program
{
    // ...
}
  • Každý program musí být ve třídě
  • Zde se jmenuje Program

3. Hlavní metoda

static void Main()
{
    // ...
}
  • Vstupní bod aplikace
  • static - nevyžaduje instanci třídy
  • void - nevrací žádnou hodnotu

4. Výpis do konzole

Console.WriteLine("Ahoj světe!");
  • Vypíše text do konzole
  • Automaticky přidá nový řádek

5. Čekání na klávesu

Console.ReadKey();
  • Zastaví program, dokud uživatel nestiskne klávesu
  • Užitečné při testování v konzoli

Celý program

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Ahoj světe!");
        Console.ReadKey();
    }
}

Proměnné

Proměnná

  • uložení dat v programu
  • data se ukládají v operační paměti (RAM)
  • musí mít určené jméno a datový typ
  • její hodnotu lze měnit
  • deklarace

Jméno proměnné

  • používáme na více místech v kódu
  • mělo by být výstižné a krátké
  • používáme zejména anglické názvy
  • jména mohou v jednom bloku existovat pouze jednou

Co je to vlastně ten blok?

using System;

class Program
{
    static void Main()
    {
        // První blok
        {
            int cislo = 10;
            Console.WriteLine("První blok: " + cislo);
        }

        // Druhý blok
        {
            int cislo = 20;  // Tady je to OK - jiný blok
            Console.WriteLine("Druhý blok: " + cislo);
        }
    }
}

Datové typy

Datový typ

  • Určení, co daná proměnná obsahuje
  • Píše se vždy před jméno proměnné
  • Ukládáme jako binární hodnotu

Celé číslo

  • short, int, long
  • short: -32 768 až +32 767
  • int: -2 147 483 648 až +2 147 483 647
  • long: -9 223 372 036 854 775 808 až +9 223 372 036 854 775 807

Celé číslo

short age = 18;
int accountBalance = 1180040;
long nationalDebt = 9223372036854775807;

Kladné celé číslo

  • ushort, uint, ulong
  • Větší rozsah, protože neobsahují záporná čísla
  • ushort: 0 až +65 535
  • uint: 0 až +4 294 967 295
  • ulong: 0 až +18 446 744 073 709 551 615

Kladné celé číslo

ushort age = 18;
uint iq = 102;
ulong distanceToStars = 18446744073709551615;

Znak

  • char
  • Lze uložit jen jeden znak

Znak

char sortBy = 'a';

Text

  • string
  • Řetězec znaků

Text

string name = "Jakub";
string school = "Smíchovská střední průmyslová škola a gymnázium";

Pravdivostní hodnota

  • boolean
  • Pouze dvě hodnoty: true a false

Pravdivostní hodnota

bool isStudent = true;
bool hasGraduated = false;

Číslo s desetinnou čárkou

  • float / double / decimal
  • float: přesnost cca 6 číslic
  • double: přesnost cca 15 číslic
  • decimal: přesnost cca 28 číslic

Číslo s desetinnou čárkou

float age = 18.4f;
double average = 1.20123d;
decimal pi = 3.14159265358979323846m;

Konstanty

  • Hodnota musí být definována již při deklaraci
  • Hodnotu nelze změnit
  • Speciální proměnná
  • Šetří výpočetní nároky

Konstanty

const int maxScore = 100;
maxScore = 120; // Chyba - nelze změnit

Konzole

Konzole

  • Třída Console — konzole
  • Komunikace mezi programem a uživatelem
  • Zapisování a čtení

Třída

  • Seskupení vlastností a metod
  • Vlastnost - speciální proměnná
  • Metoda - rozdělení příkazů
  • Přístup pomocí tečky (.)

Metody a funkce

  • Část kódu, kterou lze zavolat
  • Přijímá parametry
  • Vrací návratovou hodnotu, nebo nic

VYPISOVÁNÍ

  • hodnoty, které lze převést na text, můžeme vypsat
  • Console.Write() a Console.WriteLine()

VYPISOVÁNÍ

Console.WriteLine("Hello world!");

const string name = "Jakub";
Console.WriteLine(name);

int age = 67;
Console.WriteLine(age);

Console.Write(name);
Console.Write(age);

Čtení textu

  • Přečte zadaný text uživatelem až do stisknutí Enteru
  • Zadaný text můžeme uložit do proměnné

ČTENÍ TEXTU

string name;
name = Console.ReadLine();

Console.WriteLine(name);

Výrazy

  • Určeno k úpravě proměnných
  • Výsledek výrazu můžeme znovu uložit do proměnné
  • Např. matematické operace

Matematické operace čísel

  • Každý jazyk obsahuje základní matematické operace
  • Sčítání, odečítání, násobení, dělení

Matematické operace čísel

int a = 10;
int b = 3;

int soucet = a + b;
int rozdil = a - b;
int soucin = a * b;
int podil = a / b;

SPOJENÍ ŘETĚZCŮ

  • Řetězce můžeme spojovat s jinými proměnnými
  • Proměnné musí být možné přepsat do textové podoby
  • Speciální operátor +

SPOJENÍ ŘETĚZCŮ

string text = "Jmenuju se: ";
const string name = "Jakub";
int age = 7;

text = text + name + ", a je mi " + age;
